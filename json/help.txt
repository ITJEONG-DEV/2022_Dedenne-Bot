'봇' 또는 'bot' 글자가 들어간 채팅 채널에 입력된 간단한 키워드에 반응합니다.

<반응키워드>
❓ 도움말(키워드: 사용법/도움말/help)
 - 사용법을 안내합니다.
⭕ 참여(키워드: 참여/join)
 - 음성 채널에 참여합니다
❌ 나가기(키워드: 나가기/leave)
 - 음성 채널에서 나갑니다
🔎 검색(키워드: 검색/조회/search [키워드])
 - youtube에서 영상을 검색합니다.
 - 사용 예시) 검색 아이유
✅ 선택(키워드: 선택/select [번호])
 - 검색한 목록 중의 영상을 선택하여 재생목록에 추가합니다.
 - 사용 예시) 선택 3
✅ 추가(키워드: 재생/추가 [링크])
 - 유투브 링크를 이용하여 재생목록에 추가합니다.
 - 사용 예시) 재생 [링크]
🎵 재생목록(키워드: 재생목록/재생 목록/queue)
 - 현재 재생목록을 보여줍니다.
 - 사용 예시) 재생목록
⏭ 스킵(키워드: 스킵/넘기기/skip)
 - 재생중인 영상을 스킵합니다
 - 사용 예시) 스킵
⏹ 중단(키워드: 중단/stop)
 - 영상 재생을 중단합니다
 - 사용 예시) 중단
⏸ 멈춤(키워드: 멈춤/일시정지/pause)
 - 재생 중인 영상을 일시정지합니다
 - 사용 예시) 일시정지
▶ 재개(키워드: 재개/다시재생/다시 재생/resume)
 - 일시정지한 영상을 이어서 재생합니다
 - 사용 예) 다시 재생
🔀 재생목록 섞기(키워드: 셔플/섞기/shuffle)
 - 재생목록을 랜덤한 순서로 섞습니다
 - 사용 예) 셔플

🔎 로스트아크 캐릭터 정보 조회(키워드: 로아 조회)
 - 로스트아크 캐릭터의 정보를 조회합니다
 - 사용 예) 로아 조회 데덴네귀여워
💰 로스트아크 마리샵 정보 조회(키워드: 로아 마리샵)
 - 로스트아크 마리샵에서 판매중인 상품 정보를 조회합니다
 - 사용 예) 로아 마리샵
🪙 로스트아크 골드 시세 조회(키워드: 로아 골드 시세)
 - 현재 골드 및 전각 시세 순위를 조회합니다
 - 사용 예) 로아 골드 시세, 로아 전각 시세 등
📓 로스트아크 전각 시세 검색(키워드: 로아 전각 [전각명])
 - 현재 로아 전각 시세를 검색합니다
 - 사용 예) 로아 전각 타대
🚩 로스트아크 점령전 정보 조회(키워드: 로아 점령전)
 - 로스트아크 점령전 정보를 조회합니다
 - 사용 예) 로아 점령전
🔎 로스트아크 레이드 정보 조회(키워드: 로아 레이드)
 - 현재 로스트아크에 존재하는 레이드 정보와 보상을 조회합니다
 - 아르고스/발탄/비아키스/쿠크세이튼/아브렐슈드/카양겔/일리아칸 지원
 - 사용 예) 로아 레이드, 로아 군단장, 로아 아르고스, 로아 발탄, 로아 비아키스 등
🔎 로스트아크 레이드 공략 조회(키워드: 로아 [레이드명] 공략) 🚧개발중🚧
 - 현재 로스트아크에 존재하는 레이드 공략(준비중) 및 배틀 아이템(구현완료)을 조회합니다
 - 아르고스/발탄/비아키스/쿠크세이튼/아브렐슈드/카양겔/일리아칸 지원
 - 사용 예) 로아 알고 배템
